# Article-iOS
A native iOS app built with Swift UIKit for reading articles. Features grid/list toggle, real-time search, article detail view, and online/offline reachability handling. Supports iPhone and iPad.

Features

🚀 Animated splash screen on launch
📰 Articles list with Grid and List toggle views
🔍 Real-time search by article title
📄 Article detail screen with title, image, and description
📡 Online / Offline reachability detection and user notification
📱 Universal support for iPhone and iPad

Installation & Setup

1. Clone the Repository
git clone https://github.com/sathyaseelan-54/Article-iOS.git

2. Install Dependencies
Make sure CocoaPods is installed on your machine. If not, install it first:

sudo gem install cocoapods
pod install

3. Open the Project
  open Articles.xcworkspace Or manually open Articles.xcworkspace in Finder by double-clicking it.

4. Build & Run

In Xcode, select your target device or simulator from the scheme selector at the top.
Press ⌘ + R or click the Run button to build and launch the app.
